# HAScraper

This project automatically scrapes **HardverApro** every hour and presents the products in a cleaner, more user-friendly interface.

### Tech Stack
- **Backend**: C#
- **Frontend**: React.js
- **Database**: PostgreSQL

### Features
- Hourly automatic scraping
- Modern and improved user interface
- Easily customizable scrape target (URL)
- Configurable main prompt

## Setup

```bash
git clone https://github.com/L4rav3l/HAScraper
cd HAScraper
docker compose up -d
```

After the containers start, the application will be available at:
http://localhost:8000
